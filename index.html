Sobre Mim

Sou estudante de Engenharia de Controle e Automação pela Universidade Federal de Pernambuco e estudante técnico pela ETE - Cícero Dias. Estudo sobre as áreas de Redes de Comunicação Industriais, programação de CLP'S (Controladores Lógicos Programáveis), aplicação com microcontroladores, e desenvolvimento eletrônico e programação de robôs de combate.

Conhecimentos

Área de Controle e Automação

  • Redes de Comunicação Industrial: Modbus-RTU, Modbus-TCP/IP e MQTT
  • Programação de CLP's: Linguagem Ladder, SCL e Grafcet em CLP's Siemens
  • Aplicação com Microcontroladores: Desenvolvimento de aplicações voltadas para área de automação residencial (Domótica) utilizando linguagem C++ e microcontroladores Esp-32
  • Robôs de Combate: Prótotipação de robôs de classe cupim, Prótotipação de PCB eletrônica para controle, linguagem C++ para controle lógico e comunicação via rádio e transmissor de aeromodelo, desenvolvimento de aplicação em React Native.

    • Área de Desenvolvimento

  • Front-end: Conhecimentos em desenvolvimento utilizando HTML5, CSS3, JavaScript e React
  • Desenvolvimento Mobile: Desenvolvimento mobile de aplicações voltada para microcontroladores utilizando JavaScript e React Native para controle de automações residencias e controle de robôs de combate utilizando protocolos UDP/TCP e Bluetooth.
